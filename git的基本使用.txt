在github上new repository(新建仓库)
在本地项目根目录初始化本地仓库

$ git init  //初始化

$ git status //查看状态是否提交到本地仓库

$ git add .  //提交全部到本地仓库($ git add 文件名  //提交单个文件)

$ git commit -m  //不加-m是提交   加-m是描述本次提交的内容

$ git remote add origin http://github.com/jfs126com/项目名.git  

$ git push -u origin master  //首次push到远程仓库加-u


命令的使用

$ git clone  github上的地址。
